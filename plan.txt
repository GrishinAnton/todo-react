todos
Функционал
    Добавление тодо
    Редактирование добавленного тодо
    Отмачение о выполнение тодо
    При двойном клике редактирование тодо
    Списко всех тодо
    Выполненных тодо
    Завершенных тодо
    Очищение списка завершенных тодо
    Галочка завершающая все тодо
    Есть крестик удаления задачи из списка
    Есть счетчик оставлшихся задач
    Все данные сохраняются в локалсторедж


Список задач это Список ul+li

Сама задача это label + input + button
При двойном клике к лейблу подключается второй инпут в которой вводятся новые данные, потом они прокидываются в лейбл
